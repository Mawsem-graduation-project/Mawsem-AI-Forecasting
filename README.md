# 🌙 Mawsem AI Forecasting Engine

## Project Overview
This repository contains the AI component of the **Mawsem** project. It focuses on predicting sales demand in the Saudi market by linking Gregorian dates to the **Hijri Calendar** and local seasons.

## Key Features
* **Hijri Integration:** Automated conversion to the UmmAlQura calendar.
* **Saudi Seasonality:** Detection of Ramadan, Eid, and Hajj seasons.
* **Forecasting Model:** Implementation of **SARIMAX** for time-series prediction with exogenous variables.
* **Visualization:** Detailed plots showing actual vs. predicted sales trends.

## Technologies Used
* Python 3.x
* Pandas & NumPy (Data Processing)
* UmmAlQura (Hijri Logic)
* Statsmodels (SARIMAX Implementation)
* Matplotlib (Data Visualization)
